```bash
bun add @porulle/core @porulle/adapter-postgres @porulle/adapter-stripe
```

## Versioning

Every package above is published in lockstep — a Patch / Minor / Major bump on one bumps all of them. This is enforced by the [Changesets](https://github.com/changesets/changesets) `fixed` group in `.changeset/config.json`.

To propose a version bump from a contributor PR:

```bash
bun run changeset
```

Pick the affected packages, the bump kind, and write a one-line summary. The `release` script handles the rest at merge time:

```bash
bun run release
```

Source is in [`packages/`](https://github.com/asyncdotengineering/porulle/tree/main/packages) on GitHub.
