Tutorials are learning-oriented. Each one walks you through building something real, step by step, so you develop practical skills with the engine.

| Tutorial | What you will learn | Time |
|----------|---------------------|------|
| [Your First Store](/tutorials/first-store/) | Config, products, inventory, cart-to-checkout flow | ~15 min |
| [Build a Loyalty Plugin](/tutorials/build-a-plugin/) | Plugin system, hooks, custom routes, database-backed schema | ~25 min |
| [Tea Shop POS](/tutorials/tea-shop-pos/) | POS plugin, terminals, shifts, split payment, Z-reports | ~20 min |

Complete [Your First Store](/tutorials/first-store/) before the others — the later tutorials build on the running store it creates.
